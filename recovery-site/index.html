Recovery-Anleitung

  • Container kaputt? docker compose down && docker compose up -d
  • VM kaputt? Aus NAS-Snapshot wiederherstellen oder bootstrap.sh ausfuehren
  • Host tot? Neue VM aufsetzen, Secrets aus Passwort-Manager, bootstrap.sh

Vollstaendige Anleitung: Recovery-Dokumentation auf GitHub

Wo finde ich die Secrets?

Secrets werden NIEMALS auf dieser Seite angezeigt.
  • Pi-hole Webpasswort: Im Passwort-Manager unter "Homelab / Pi-hole"
  • GitHub PAT (nur Master): Passwort-Manager unter "Homelab / GitHub PAT"
  • SSH Deploy-Key (nur Follower): Passwort-Manager unter "Homelab / Deploy Key"

Bei Recovery: Secrets manuell aus dem Passwort-Manager auf die VM kopieren. Siehe docs/recovery.md fuer die genauen Pfade und Berechtigungen.
